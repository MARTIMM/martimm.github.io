The Gnome::Gtk4::R-ConstraintTarget interface is implemented by objects that can be used as source or target in Gnome::Gtk4::Constraints.

Besides Gnome::Gtk4::Widget, it is also implemented by Gnome::Gtk4::ConstraintGuide.
