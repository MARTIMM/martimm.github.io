new-cclosureexpression

Creates a Gnome::Gtk4::Expression that calls callback_func when it is evaluated.

This function is a variant of .newcclosureexpression() in class Gnome::Gtk4::ClosureExpression that creates a Gnome::GObject::N-Closure by calling g_cclosure_new() with the given callback_func, user_data and user_destroy.

method new-cclosureexpression ( GType $value-type, GClosureMarshal &marshal, UInt() $n-params, N-Object() $params, GCallback &callback-func, gpointer $user-data, GClosureNotify &user-destroy --> Gnome::Gtk4::CClosureExpression )

  • $value-type; the type of the value that this expression evaluates to.

  • GClosureMarshal &marshal; marshaller used for creating a closure. The function must be specified with the following signature; :( N-Object $closure, N-Object $return-value, guint $n-param-values, N-Object $param-values, gpointer $invocation-hint, gpointer $marshal-data ).

  • $n-params; the number of params needed for evaluating $closure.

  • $params; (transfer ownership: full) expressions for each parameter.

  • GCallback &callback-func; callback used for creating a closure. The function must be specified with the following signature; :( ).

  • $user-data; user data used for creating a closure.

  • GClosureNotify &user-destroy; destroy notify for $user-data. The function must be specified with the following signature; :( gpointer $data, N-Object $closure ).
