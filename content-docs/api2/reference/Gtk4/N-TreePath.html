new-from-string

Note: The native version of this routine is deprecated in gtk4-lib() since version 4.10

Creates a new Gnome::Gtk4::N-TreePath initialized to $path. $path is expected to be a colon separated list of numbers. For example, the string “10:4:0” would create a path of depth 3 pointing to the 11th child of the root node, the 5th child of that 11th child, and the 1st child of that 5th child. If an invalid path string is passed in, undefined is returned.

method new-from-string ( Str $path --> Gnome::Gtk4::TreePath \)

  • $path; The string representation of a path.
